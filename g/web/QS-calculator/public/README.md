# Public Assets

## Icons Required

Please add the following icon files:

1. **icon-192.png** - 192x192 app icon
2. **icon-512.png** - 512x512 app icon
3. **badge.png** - Badge for notifications
4. **screenshot1.png** - App screenshot for PWA

### Creating Icons

Use a tool like:
- [Favicon.io](https://favicon.io/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- Photoshop/Figma

### Design Suggestions

- Primary color: #2563eb (blue)
- Include: Ruler, calculator, or construction symbol
- Keep simple and recognizable at small sizes

### Temporary Solution

For now, you can use Font Awesome icons as placeholders in the browser.
